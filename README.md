# prueba-tecnica-analisis-datos
Repositorio de prueba técnica para AtentaMente
Este repositorio contiene el análisis y limpieza de una base de datos correspondiente a la asistencia de participantes a cinco sesiones de un curso.

La base original es la unión de múltiples fuentes, por lo que presenta inconsistencias como registros duplicados, valores faltantes y diferencias en el formato de nombres y correos electrónicos.

El objetivo principal es obtener una base de participantes únicos, consolidar su asistencia y analizar su desempeño.
Se solicitó y obtuvo autorización para realizar la prueba técnica utilizando Python en lugar de R.

Por este motivo, el análisis se desarrolla en un Jupyter Notebook (.ipynb), empleando principalmente la librería pandas para la manipulación de datos.
